# 学生信息管理系统 python1147

本项目是基于 Flask 开发的学生信息管理系统，集成了教学管理功能，致力于提高学校管理的效率和准确性。

## 技术栈

- Python
- Flask
- Navicate（可能是指 Navicat，数据管理工具）
- MySQL
- PyCharm

## 功能模块

- **用户模块**
  - 登录
  - 注册
- **学生管理**
  - 学生信息管理
  - 成绩管理
- **班级管理**
  - 班级信息管理
- **课程管理**
  - 课程信息管理
- **宿舍管理**
  - 宿舍信息管理
- **任务管理**
  - 任务发布与跟踪
- **资料管理**
  - 教学资料的上传与管理
- **反馈模块**
  - 意见反馈
- **系统管理**
  - 系统设置与维护

## 系统角色

- 学生
- 教师
- 管理员

## 运行环境

- Python 3.x
- Flask 1.x
- MySQL 5.x

## 部署步骤

1. 环境准备：安装 Python、Flask、MySQL
2. 数据库配置：使用 Navicate 创建数据库及表结构
3. 应用部署：配置 Flask 应用及数据库连接
4. 启动服务：运行 Flask 应用
